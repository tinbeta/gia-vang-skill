# gia-vang

Skill để lấy giá vàng từ PNJ (Phu Nhuan Jewelry) - nhà bán lẻ vàng hàng đầu Việt Nam.

## API

API key không cần thiết, sử dụng API công khai của PNJ.

## Cài đặt

Không cần cài đặt thêm, chỉ cần Python 3 có sẵn.

## Sử dụng

```bash
python3 scripts/gia_vang.py
```

## Tính năng

- Lấy giá vàng PNJ và SJC theo thởi gian thực
- Hỗ trợ các khu vực: TPHCM, Hà Nội, Đà Nẵng, Cần Thơ
- Tùy chọn lọc theo loại vàng hoặc khu vực
